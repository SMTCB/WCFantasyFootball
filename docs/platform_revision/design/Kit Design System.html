01 · Foundation

Color, Type & Spacing Tokens

02 · Shell & Navigation

App Shell & Navigation Patterns

The defining structural decision: a dark navy sidebar anchors the chrome. Warm white fills the content area. No dual CTA bars inside tabs. Mobile uses a bottom tab bar — the sidebar is replaced entirely, not overlaid.

05 · White Label

3-Token Client Re-skin

A client override requires changing exactly three values. Everything else — typography, spacing, component shapes, layout patterns — stays constant. The system was designed for this from the ground up.
