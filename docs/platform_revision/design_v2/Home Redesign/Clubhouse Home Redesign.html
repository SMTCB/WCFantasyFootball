Frontrow — Clubhouse Home design proposal

Kit Light · reconciled tokens · desktop 1440 & mobile 390

The composition call

Home, Recap and Chat merge into one Discord/Slack-style view: a main column (competitions grid + activity feed, Recap folded in) alongside a persistent chat rail — always visible, never a tab you leave to check. The Forza Times keeps its own broadsheet identity (deliberate) but surfaces as a teaser card inside the main column.

Eight tabs become five — Home · The FrontRow · Members · Find · Settings. Inbox drops from a tab to a notification panel in the header. Chat fully merges into Home on desktop; on mobile, where there's no room for a rail, it's its own screen (S-06).

Sidebar & multi-Clubhouse switching

The switcher moves into the sidebar as a row of identity circles beneath the brand mark — one click from anywhere. A single-Clubhouse user never sees the row; a five-Clubhouse user gets a wrapping grid. The in-page pill row is removed.

Top bar — no active competition

On Home, a dedicated 🏠 Clubhouse pill leads the competition strip and takes the accent-active treatment. Inside a competition, the pattern flips.

Update — rebrand + new screens

Platform renamed FantasyKit → Frontrow ("Your seat to all the action") — the Tiers logo mark now replaces the placeholder wordmark in the sidebar. Added since the last round: Members (S-05), a full-width Chat screen (S-06), the New Competition flow using the real sport marks (S-07), and a redesigned Notifications panel replacing the emoji bell (S-08). Every screen below now ships as a desktop + mobile pair with a code and description for handoff.
