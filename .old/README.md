## When to Use This Folder

**Don't** look here during normal development. These files are here for **historical reference only**:
- If you want to understand previous design thinking or decisions
- If you need to research why something was chosen or rejected
- If you're documenting project history

**Do** reference active documentation in:
- `docs/` — Current, organized project documentation
- `.working-docs/` — Session and behavioral instructions
- `CLAUDE.md`, `BACKLOG.md`, `README.md` — Root-level active files

## Cleanup

Files here are not tracked as closely. If any of these are genuinely no longer needed (e.g., 6 months from now), they can be deleted without hesitation. The git history preserves them anyway.
