## Brand Assets

### 1. Editorial Brandmark
**File**: `public/brandmark.svg`

Logo with stacked "FORZA / FANTASY / LEAGUE" layout:
- FORZA in large, italic Archivo Black
- Aligned slash (3px, 15° rotation, cyan)
- FANTASY / LEAGUE in stacked columns

**Use**: Website headers, pitch decks, marketing materials

### 2. Tactical Icon
**File**: `public/tactical-icon.svg`

Three-line tactical formation (FWD/MID/DEF):
- Cyan (top) → Paper (middle) → Mute (bottom)
- Represents squad formation and strategy

**Use**: 
- App icon (iOS/Android)
- Favicon
- UI shorthand
- Marketing assets

### 3. App Icons
**Generated by**: `npm run icons:generate`

Automatically generates:
- **iOS**: 17 icon sizes (40×40 → 1024×1024) in `ios/App/App/Assets.xcassets/AppIcon.appiconset/`
- **Android**: 6 densities (ldpi → xxxhdpi) in `android/app/src/main/res/mipmap-*/`

The tactical icon is scaled and positioned within a dark rounded square background.

### 4. BrandMark Component
**File**: `src/components/BrandMark.jsx`

React component that renders the editorial brandmark with configurable:
- `theme`: 'dark' (default) | 'light'
- `scale`: multiplier (default 1)
- `compact`: boolean for single-line layout

**Integrated in**:
- `src/components/AppLayout.jsx` — sidebar navigation header
- `src/screens/AuthScreen.jsx` — login/signup page header

## Next Steps

### Before Store Submission

1. **Generate App Icons**
   ```bash
   npm install  # installs sharp if not present
   npm run icons:generate
   ```

2. **Review Icons**
   - Open `ios/App/App.xcodeproj` in Xcode → Assets.xcassets
   - Open `android/` in Android Studio → mipmap folders
   - Verify all sizes are present and rendering correctly

3. **Sync to Native**
   ```bash
   npm run native:sync  # builds web + syncs Capacitor
   npx cap open ios     # review in Xcode
   npx cap open android # review in Android Studio
   ```

4. **Update Store Listings**
   - App Store: App Name, Subtitle, Description, Keywords, Category
   - Play Store: App title, Short description, Full description, Category
   - Both: Screenshots, Preview video (optional)

5. **Legal & Compliance**
   - Privacy Policy URL (required)
   - Terms of Service (recommended)
   - Support email/contact
   - Age rating questionnaires

### Brand Usage Guidelines

✅ **Do**:
- Use official names: "Forza Fantasy League" or "Forza"
- Pair logo with supporting text on small spaces
- Apply cyan accent sparingly for CTAs
- Maintain generous whitespace around brandmark

❌ **Don't**:
- Distort or rotate the brandmark
- Change colors without approval
- Mix with old "FantasyKit" or "ForzaKit" names
- Use unauthorized fonts
